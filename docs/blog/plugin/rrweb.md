# rrweb -- 录制并回放 web 界面用户操作

`rrweb` 全称 `Record and replay the web`，指利用浏览器提供的 `API` 实现对用户在 `web` 页面中的任意操作进行录制与回放

本质上来说，并非是通过录制视频来记录用户的操作，也不是通过播放序列帧来回放用户操作，而是通过记录用户的操作序列（用户行为、DOM 变化），在回放时将操作序列重新执行一遍
